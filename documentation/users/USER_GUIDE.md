Welcome to the Smart Enterprise Suite (SES). This guide will help you navigate the system and perform daily tasks.

### Valid Credentials

2. Enter your **Email** and **Password**.
3. Click **Login**.

## üõ†Ô∏è Maintenance Requests

### Creating a New Request
1. Go to the **Maintenance** tab.
2. Click **+ New Request**.
3. Enter Customer details (or search for existing).
4. Select the **Machine** (Serial Number).
5. Enter the **Complaint** details.
6. Click **Submit**.

### Tracking Status
- **Open**: Request received, waiting for technician.
- **In Progress**: Technician is working on it.
- **Waiting Approval**: High-cost repair needs manager approval.
- **Closed**: Repair complete and delivered.

## üì¶ Machine & Inventory

### Checking Stock
1. Go to **Warehouse** > **Inventory**.
2. Use the search bar to find parts.
3. Check the **Quantity** column.

### Machine History
You can view the full repair history of any machine:
1. Go to **Customers** or **Machines**.
2. Search for the serial number.
3. Click the **History** icon.

1. When closing a request, if there is a cost, a payment prompt will appear.

3. Enter the amount.
4. A receipt will be generated automatically.
